## Description
Nest js Demo Project

## Node Version
Node Version - [20.9.0] - Sequelize Only supports upto this version.


## Project setup

```bash
$ yarn install
```

## Database
```bash
$ npm i -g sequelize-cli
```

```bash
$ npx sequelize-cli db:migrate
```

```bash
$ npx sequelize-cli db:seed:all
```

## Test

```bash
$ npm run test
```


## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test
